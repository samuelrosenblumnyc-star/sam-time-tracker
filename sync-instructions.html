SAM TIME TRACKER

Cross-Device Sync Setup

Your time tracker stores data in a private GitHub Gist — a free, secure snippet of JSON tied to your GitHub account. From any device, you can push your latest data up or pull the newest version down with a single button click in the app's Settings tab.

Before you start: You need a free GitHub account. If you don't have one, sign up at github.com. The sync uses only a single file in a private Gist — it is never publicly visible.

Step 1 — Create a Personal Access Token

Do this once, on any device

  1. Sign in to github.com.
  2. Click your profile picture (top-right) → Settings.
  3. In the left sidebar, scroll to the bottom and click Developer settings.
  4. Click Personal access tokens → Tokens (classic).
  5. Click Generate new token → Generate new token (classic).
  6. Give it a name like sam-time-tracker and set an expiry (or choose "No expiration").
  7. Under Select scopes, check only gist.
  8. Click Generate token.
  9. Copy the token immediately — it starts with ghp_ and is only shown once.

Step 2 — Push from Your First Device

First-time setup

  1. Open SAM TIME TRACKER in your browser.
  2. Click the Settings tab.
  3. Scroll down to the Cross-Device Sync section.
  4. Paste your token into the GitHub Personal Access Token field.
  5. Leave the Gist ID field blank (it will be created automatically).
  6. Click ↑ Push to Gist.
  7. After a moment, a Gist ID appears in the field — copy it. You'll need it on other devices.

Step 3 — Pull on Each Additional Device

Repeat for every other device

  1. Open SAM TIME TRACKER in the browser on the new device.
  2. Go to Settings → Cross-Device Sync.
  3. Paste the same GitHub Personal Access Token.
  4. Paste the Gist ID you copied in Step 2.
  5. Click ↓ Pull from Gist and confirm.
  6. All your data loads instantly. Done.

Day-to-Day Usage

  • Switching devices: Push on your current device, then Pull on the next one.
  • The token and Gist ID are saved in each browser after you enter them once — you won't need to re-enter them.
  • Pull overwrites local data with the remote version, so always Push your latest changes before switching.
  • All data is synced: time tracker entries, to-do lists, saved quick-pick items, and settings.
Security note: Your token is stored only in your browser's local storage and sent exclusively to the GitHub API over HTTPS. It is never logged or transmitted anywhere else. If you ever want to revoke access, delete the token from your GitHub Developer Settings.
